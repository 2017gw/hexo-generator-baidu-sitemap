# Sitemap generator

Generate sitemap.

## Install

``` bash
$ npm install hexo-generator-baidu-sitemap --save
```

## Update

``` bash
$ npm remove hexo-generator-baidu-sitemap --save
$ npm install hexo-generator-baidu-sitemap --save
```

## Options

You can configure this plugin in `_config.yml`.

``` yaml
baidusitemap:
    path: baidusitemap.xml
```

- **path** - Sitemap path. (Default: baidusitemap.xml)
